  • Project date: 01 June, 2018

Overview:

Based on many girls experience, they had no idea what kind of career we can gain from technology. There are so many possibilities that don't cross in our heads. Therefore, I decided to create an interactive game that educates young women on different tech careers.

How I built it

As a group, we decided to combine all our technical skills to create different games and stories for each character. We used JQuery, Bootstrap, Photoshop, HTML, CSS, javascript, angular.js, and Google Cloud Platform.
